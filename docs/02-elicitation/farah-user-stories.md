# User Stories

The following user stories are based on the stakeholders I identified from the REMS scenario, focusing on improving clarity, efficiency, and overall workflow.



## Administrative Staff

- As an administrative staff member, I want to verify that an application is complete before assigning it for review, so that incomplete submissions do not cause delays in the process.

- As an administrative staff member, I want to track the status of all applications in one place, so that I can manage the workflow more efficiently and avoid losing track of progress.



## Researchers

- As a researcher, I want to receive clear and structured feedback on my application, so that I can understand what needs to be revised and respond effectively.



## Ethics Committee Members (Reviewers)

- As a committee member, I want to access all relevant documents in one place, so that I can review applications more efficiently without searching across multiple sources.
