# REMS Project – User Stories

## Administrative Staff

1. **As an administrative staff member, I want to check submitted applications for completeness so that incomplete applications can be identified early.**

2. **As an administrative staff member, I want to track the progress of applications so that I can manage workflow efficiently.**

3. **As an administrative staff member, I want to manage communication between researchers and reviewers so that the review process runs smoothly.**

---

## Researcher

4. **As a researcher, I want to complete all required fields and upload necessary documents so that I can successfully submit my application.**

5. **As a researcher, I want to be guided through the application process so that I do not miss any required steps.**

6. **As a researcher, I want to view the status of my application so that I can track its progress.**

---

## Ethics Committee Member

7. **As a committee member, I want to view applications assigned to me so that I can review them efficiently.**

8. **As a committee member, I want to communicate with researchers so that I can request clarifications when needed.**
