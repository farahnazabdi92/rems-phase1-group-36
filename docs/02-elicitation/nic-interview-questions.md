### Selected stakeholder: Administrative staff

**Question 1:**
As of now, how do you track the status of applications?

**Uncertainty, assumption, gap:**
We were told that staffs use spreadsheets to track applications but it is unclear on how they manage workflow or updates.

**How it refine requirements:**
This helps explain the requirements for tracking application status. Displaying current status and notifying pending tasks are features that should be added.

**Question 2:**
What do you do if an application is not complete or requires more information?

**Uncertainty, assumption, gap:**
Details on communication between staffs and researchers are incomplete. It is unclear how staffs track which applications need follow-up, how they notify researchers, and how they ensure responses are received.

**How it refine requirements:**
Helps define system behavior for handling incomplete applications. Features like automated responses to researchers and notifications for staffs once the applications are completed should be implemented.
