**Researchers**
1. As a researcher, I want to submit my application and all required supporting documents through the system, so that my submission is complete in one go.
2. As a researcher, I want to upload new versions of documents without overwriting old ones, so that I can track any changes over time.

**Administrative staffs**
1. As an administrative staff, I want to return incomplete applications to researchers, so that they can revise and resubmit them again.
2. As an administrative staff, I want to view the audit trail of an application, so that I can track its history and status.

**Committee members**
1. As a committee member, I want to be informed of the members of a research team beforehand, so that I can better understand the structure of the team.
2. As a committee member, I want to declare and manage conflicts of interest, so that I can avoid participating in biased decision-making.
